# Kita-Kalender

Dieses Paket ist für GitHub Pages oder einen anderen Webhost gedacht.

## Enthalten
- `Kita_Lummerland.ics` – Kalenderfeed für Apple Kalender, Outlook und andere Kalender-Apps
- `index.html` – kleine Landingpage zum Abonnieren
- `README.md` – kurze Anleitung

## So nutzt du es auf GitHub Pages
1. Neues öffentliches GitHub-Repository anlegen, z. B. `kita-kalender`.
2. Diese Dateien ins Repository hochladen.
3. In den Repository-Einstellungen unter **Pages** GitHub Pages aktivieren.
4. Als Quelle den Haupt-Branch und den Root-Ordner wählen.
5. Danach ist die Datei über eine URL wie diese erreichbar:
   `https://DEIN-USERNAME.github.io/kita-kalender/Kita_Lummerland.ics`

## Wichtig für Apple Kalender
Apple Kalender braucht einen direkten Link auf die `.ics`-Datei. Die URL darf keine Vorschauseite sein, sondern muss direkt die Datei liefern.

## QR-Code
Der beiliegende QR-Code zeigt auf eine Platzhalter-URL. Sobald die echte GitHub-Pages-URL feststeht, den QR-Code neu erzeugen.

## Aktualisierung
Wenn der neue Jahreskalender da ist:
- `Kita_Lummerland.ics` ersetzen
- committen
- fertig

Alle Abonnenten erhalten die Aktualisierung automatisch beim nächsten Abruf.
