## Dateien
- `index.html` – Landingpage für Eltern
- `Kita_Lummerland.ics` – Kalenderabo
- `qr-code.png` – QR-Code zur Startseite

## Upload-Reihenfolge
1. `index.html` hochladen
2. `Kita_Lummerland.ics` hochladen
3. `qr-code.png` hochladen
4. Auf GitHub Pages warten, bis die Seite aktualisiert ist

## Hinweis
Der Button „Kalender abonnieren“ verwendet `webcal://` für Apple-Geräte.
Die direkte Datei bleibt unter der HTTPS-URL erreichbar.
