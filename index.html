Kita-Kalender abonnieren

Scannen Sie den QR-Code oder öffnen Sie den Kalender-Link, um die aktuellen Kita-Termine direkt im eigenen Kalender zu sehen. Änderungen werden automatisch übernommen.

🍎 Kalender abonnieren 📄 ICS-Datei öffnen 🌐 Diese Seite teilen
Für Apple Geräte Ein Klick auf den Button öffnet direkt das Kalenderabo.
Für alle Eltern Auch Android, Outlook und andere Kalender-Apps können den Link nutzen.
Automatische Updates Wenn die ICS-Datei aktualisiert wird, erscheinen die neuen Termine automatisch.

QR-Code

Scannen Sie den QR-Code mit dem Smartphone. Er führt zur Abo-Seite mit dem Kalenderlink.

So funktioniert es

1. QR-Code scannen

Der Code öffnet diese Seite mit dem Kalenderabo.

2. Auf „Kalender abonnieren“ tippen

Apple Kalender öffnet sich mit dem Abo-Dialog.

3. Abo bestätigen

Die Termine werden in den Kalender übernommen und aktuell gehalten.

4. Seite später erneut nutzen

Die URL bleibt gleich. Nur die ICS-Datei wird bei Bedarf ersetzt.
